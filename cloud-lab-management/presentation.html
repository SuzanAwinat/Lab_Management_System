User Roles & Permissions

Role Hierarchy

  • System Administrator: Full system access, user management, global settings
  • Campus Administrator: Campus-level management, budget allocation, reporting
  • Lab Manager: Lab operations, inventory management, booking approvals
  • Instructor: Create bookings, view schedules, request resources
  • Student: View schedules, limited booking capabilities

Permission Matrix

Granular Permissions:

  • Inventory: View, Add, Edit, Delete, Approve
  • Bookings: Create, Modify, Cancel, Approve
  • Budget: View, Allocate, Approve Expenses
  • Reports: View, Generate, Export
  • Users: View, Create, Modify, Deactivate
